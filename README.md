# RexUrl

## Setup SQLite version
1. install sqlite3
```
npm install sqlite3
```
2. create database
```
open db_rexurl.db
```
3. alter .env.sample and /models/db.js connection function

4. run service
```
npm run watch
```

## Setup MySQL version
1. install package
```
npm install
```
2. pull the MySQL image
```
docker pull mysql
```
